== Description ==

This plugin allows you to create links to elements of the media in the Wysiwyg editor when you click the link button.

== Installation ==

1. Upload `deefuse-link-media-in-wysiwyg` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
